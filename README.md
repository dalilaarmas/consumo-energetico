# Consumo Energético

Proyecto backend en Java para el análisis de consumos energéticos de edificios.

Los datos se migran desde archivos JSON a una base de datos MySQL utilizando Flyway.

## Tecnologías

- Java
- MySQL
- Maven
- Flyway
- JDBC

## Estructura del proyecto

dao → acceso a base de datos  
entity → modelos de datos  
db → conexión a la base de datos

## Base de datos

La base de datos utilizada es **MySQL**.  
Las migraciones de esquema se gestionan con **Flyway**.
